# Roadmap

This roadmap describes possible improvements for future versions of edHash PatternForge.

## v1.0.0

Initial public release.

Included:

- Controlled pattern generation.
- Storage estimation.
- Audit-oriented pattern mode.
- Local report generation.
- Terminal-based workflow.
- edHash branding.

## Future improvements

Planned ideas:

- Add more usage examples.
- Add fictional lab scenarios.
- Improve report formatting.
- Add optional CSV export.
- Add optional JSON export.
- Add unit tests.
- Improve terminal compatibility.
- Add configuration profiles.
- Add safer output folder management.
- Add documentation screenshots.

## Long-term vision

The long-term objective is to evolve PatternForge into a cleaner security lab utility for pattern analysis, defensive auditing and technical education.

The tool should remain:

- Local.
- Transparent.
- Documented.
- Controlled.
- Useful for learning and authorized research.
